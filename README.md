This repository contains the U-Net and NVIDIA MONAI pre-trained model for 3D segmentation of brain tumors based on BraTS data that we have tested for automated segementation accuracy using the BraTS 2021 Task 1 dataset.

U-Net: https://arxiv.org/abs/1606.06650
MONAI: https://catalog.ngc.nvidia.com/orgs/nvidia/teams/monaitoolkit/models/monai_brats_mri_segmentation?version=0.5.2
